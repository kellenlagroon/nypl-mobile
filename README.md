# NYPL Mobile App Demo

A static, interactive mobile-app concept for exploring a possible New York Public Library experience. It is built as a self-contained HTML demo and can be opened locally or hosted as a static site.

## Run locally

From the project directory, start any static file server and open `index.html` in a browser. For example:

```sh
npx serve -l 4173 .
```

Then visit `http://127.0.0.1:4173`.
